# Zario 入門

Zario は、Node.js 向けの **軽量・高速・高機能** なロギングライブラリです。TypeScript で構築されており、シンプルなアプリケーションから複雑な分散システムまで対応できる、直感的で包括的な API を提供します。

## 設計思想

Zario は、次のようなロギング体験を目指して設計されています。

- **軽量**：依存関係ゼロ、極小フットプリント
- **高パフォーマンス**：非ブロッキング非同期モードで高スループットを実現
- **柔軟**：トランスポート・フィルター・エンリッチャーによる高い拡張性
- **開発者フレンドリー**：直感的な API と TypeScript の完全サポート

## 主な機能

- **高パフォーマンス**：最適化されたシリアライズとオプションの非同期ログ出力により最小限のオーバーヘッドを実現  
  → 詳細は [Benchmarks](./benchmarks.md)
- **柔軟なフォーマット**：開発時はカラー付きコンソール出力、本番環境では構造化 JSON 出力
- **📁 プラグイン可能なトランスポート**：Console、ファイル（ローテーション・圧縮対応）、HTTP エンドポイント、または独自実装が可能
- **🧩 コンテキスト対応ログ**：子ロガーでモジュールやリクエスト単位のコンテキスト管理が可能
- **🔍 高度なフィルタリング**：レベル、プレフィックス、メタデータ、カスタム条件によるログ制御
- **🏗️ 構造化ログ**：静的・動的メタデータを付与できる拡張パイプライン
- **📊 ログ集約**：バッチ処理や時間ベース集約による効率的なログ管理

## ユースケース

### 🌐 Web サーバー & API
リクエスト、サーバーエラー、パフォーマンス指標を記録。Express、Fastify、Koa などに最適。リクエスト単位の子ロガーで処理フローを追跡できます。

### ⚡ サーバーレス関数
AWS Lambda や Vercel などの短命環境でも、コールドスタートへの影響を最小限に抑えつつログを出力できます。

### 🏗️ マイクロサービス
複数サービスのログを集約し、構造化 JSON によって ELK、Datadog、New Relic などへ簡単に連携できます。

### 🔧 CLI アプリケーション
ユーザーには見やすいカラー出力、内部では詳細なデバッグログを保持できます。

### ⚙️ バックグラウンドジョブ
長時間タスクや cron ジョブの進捗・状態を記録し、実行時間の自動計測も可能です。
