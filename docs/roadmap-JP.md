# ロードマップ

Zario は、Node.js エコシステム向けに **堅牢かつ柔軟なロギング機能** を提供するため、継続的に改善・拡張を行っています。

---

## 実装済み機能（Completed Features）

- **Console / File トランスポート**: ローテーション・圧縮対応の信頼性の高いローカルログ出力
- **HTTP トランスポート**: リトライ機能付きのリモートログ送信
- **子ロガー（Child Loggers）**: 階層的なコンテキストと設定の継承
- **非同期モード（Async Mode）**: 高パフォーマンス向けのノンブロッキング I/O
- **高度なフィルタリング**: きめ細かなログ制御を可能にする柔軟な条件指定
- **構造化ログ（Structured Logging）**: メタデータ拡張のための強力なエンリッチメントパイプライン
- **ログ集約（Aggregation）**: バッチ処理および時間ベース処理に対応

---

## 今後の予定（Future Plans）

### ネイティブ連携
- **Syslog 対応**: Unix / Linux 向けネイティブ syslog トランスポート
- **データベーストランスポート**: MongoDB / Redis / PostgreSQL への直接出力
- **Elasticsearch 連携**: バルクインデックス最適化付きトランスポート

### 新機能
- **WebSocket トランスポート**: 監視ダッシュボード向けリアルタイムログ配信
- **ログ暗号化**: 機密ログのエンドツーエンド暗号化
- **パフォーマンスメトリクス**: スループットやレイテンシを可視化する内蔵ダッシュボード
- **React Native 対応**: モバイル開発環境での利用サポート

### 最適化
- **ゼロアロケーションロギング**: GC 負荷をさらに削減するためのメモリ最適化
- **高度な検索機能**: ローカルログファイルの解析・検索ユーティリティ
